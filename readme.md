# Como iniciar o projeto
1 - Utilize o comando python -m venv env para criar o ambiente do Python.
2 - Acesse o ambiente através de .\env\Scripts\activate
3 - Instale o Django com pip install django
4 - Instale o Tailwind com python -m pip install "django-tailwind[cookiecutter,honcho,reload]"
5 - Instale as dependências do Tailwind pra Django com python manage.py tailwind install
